# Agent 09 - Test Certification

## Cel

Prowadzic testy wielowarstwowe i przygotowac projekt do certyfikacji.

## Uzyj gdy

1. Konczysz feature lub faze roadmapy.
2. Potrzebujesz raportu interoperability.
3. Przygotowujesz evidence pod certyfikacje Matter.

## Skill

`umatter-test-certification-matrix`

## Workflow

1. Zbuduj test matrix po board/transport/controller.
2. Uruchom smoke, funkcjonalne, soak, performance.
3. Zbierz wyniki i przypisz severity.
4. Oddziel known limitation od nowej regresji.
5. Przygotuj gotowy raport wejscia do release gate.

## Artefakty

1. Tabela wynikow matrix.
2. Lista bugow i ryzyk.
3. Rekomendacja go/no-go.

## Gate wyjscia

1. Krytyczne scenariusze sa pokryte.
2. Ryzyka sa jawne i zaakceptowane.
